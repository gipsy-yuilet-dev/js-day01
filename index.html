operadores de comparacion:
a continuacion una explicacion de los operadores de comparacion de JS:

    1. Mayor que : > este operador la funcion es si el valor de la variable a es mayor que el valor de la variable b, entonces sera verdadero, de lo contrario sera falso
      ejemplo: a = 5, b = 3, a > b, entonces sera verdadero
    2. Menor que: < este operador La funcion es si el valor de la variable a es menor que el valor de la variable b entonces sera verdadero, de lo contrario sera falso ejemplo: a = 3, b = 5, a < b, entonces sera verdadero
    3. Mayor o igual que: >= En este caso el operador la funcion es si el valor de la variable a es mayor o igual que el valor de la variable b, entonces sera verdadero, de lo contrario sera falso ejemplo: a = 5, b = 5, a >= b, entonces sera verdadero
    4. Menor o igual que: <= Efecto contrario al operador Mayor o igual que, si variable a es menor o igual que el valor de la variable b, entonces sera verdadero, de lo contrario sera falso
      ejemplo: a = 3, b = 5, a <= b, entonces sera verdadero
    5. Diferente :!= este operador, La funcion es si el valor de la variable a es diferente al valor de la variable b, entonces sera verdadero, de lo contrario sera falso ejemplo: a = 5, b = 3, a != b, entonces sera verdadero
    6. Igual a: == este es un simbolo de igual que muchas veces se puede confundir con identico a ==, la funcion es si el valor de la variable a es igual al valor de la variable b, entonces sera verdadero, de lo contrario sera falso ejemplo: a = 5, b = 5, a == b, entonces sera verdadero
    7. Identico a: === compara si el valor de la variable a es identico al valor de la variable b y tambien compara si el tipo de dato de la variable a es igual al tipo de dato de la variable b, entonces sera verdadero, de lo contrario sera falso
      ejemplo: a = 5, b = 5, a === b, entonces sera verdadero
    8. No identico a: !== este operador la funcion es si el valor de la variable a no es identico al valor de la variable b entonces sera verdadero, de lo contrario sera falso ejemplo: a = 5, b = 3, a !== b, entonces sera verdadero
